# Project manager

Use The Agentsy project manager to organize the current work.

Target agent: `agentsy-project-manager`

Steps:

1. Read `.cursor/agents/agentsy-project-manager.md`.
2. Restate the current brief in one or two sentences.
3. Track staff, decisions, assumptions, blockers, risks, and open questions.
4. Compare the current work against the definition of done.
5. End with the next three concrete actions.

Return:

- Brief status
- Decision log
- Open questions or blockers
- Staff tracker
- Definition-of-done check
- Next actions

If the user included task details after the slash command, use those details as the brief.
